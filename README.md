# kmaster

A tool to securely store personal keys and passwords

## Server
- sign-in / sign-up
    - email
    - password
    - fingerprint
- update / push
    - email
    - json

## Client
- list labels
- set
    - label
    - description
    - username
    - password
        - auto
        - manual
    - public-key
    - private-key
    - custom
        - key
        - value
- update(label)
- display(label)
    - label
    - description
    - username
    - custom keys
- copy(label)
    - username
    - password
    - public-key
    - private-key
    - custom(key)
        - value
